## Ecommerce APP in React
 
### Features
- Display the Product list from the Api
- On Click of particular Product show its description
- Allow user to search the products
- Applied routing for routing to Home and ProductsList Page
---

## How to Run

1. **Clone the repository**
   ```bash
   git clone https://github.com/Jenith-Panchal-Simform/Practical-06-Ecommerce.git
   cd Practical-06-Ecommerce
   ```
2. **Install Dependencies**
   ```bash
   npm install
   ```
3. **Start Development Server**
   ```bash
   npm run dev
   ```
